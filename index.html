Earth as Art

I found the Earth as Art page at NASA. At first it didn't grab my attention, until I saw the Flowing Data post.

I looked through the Earth as Art PDF, since I don't have an iPad anymore I couldn't use the iPad App.

Next, I decided to deploy Earth as Art, as if I was running the project. I deployed ot first as a Github repository, then generated a JSON data store for all images, titles, descriptions and location.

I decide to deploy a web site using Github Pages + Mustache templates to display the satellite imagery as HTML files. Next I will use the JSON and images to render presentations and other interactive implementations that allow the art to be viewed, shared and syndicated.
