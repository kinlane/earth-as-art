Earth as Art

This is a project derived from the NASA, Earth as Art photo collection.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to GitHub Pages.

I found the Earth as Art page at NASA. At first it didn't grab my attention, until I saw the Flowing Data post.

After I got a taste, I looked through the Earth as Art PDF, since I don't have an iPad anymore I couldn't use the iPad App. Of course I wanted more...

So I decided to deploy Earth as Art, as if I was running the project. I deployed first as a Github repository, then generated a JSON data store for all images, titles, descriptions and location.

I decide to deploy a web site using Github Pages + Mustache templates to display the satellite imagery as HTML files. Next I will use the JSON and images to render presentations and other interactive versions to enjoy the work.
